# Examples

This folder contains worked examples for the results-db skill.

## Included

- `example_estimates.csv`: toy estimate data
- `example_results_database.csv`: example results ledger output

## Workflow

1. Inspect `example_estimates.csv`
2. Run `python scripts/populate_example.py --project . --input examples/example_estimates.csv`
3. Review the generated `results/results_database.csv`
